Class Encoder

  • java.lang.Object
    • org.myrobotlab.framework.Encoder

  • public class Encoder
extends java.lang.Object
    handles all encoding and decoding of MRL messages or api(s) assumed context - services can add an assumed context as a prefix /api/returnEncoding/inputEncoding/service/method/param1/param2/ ... xmpp for example assumes (/api/string/gson)/service/method/param1/param2/ ... scheme = alpha *( alpha | digit | "+" | "-" | "." ) Components of all URIs: [:][#] http://stackoverflow.com/questions/3641722/valid-characters-for-uri-schemes
