      • moveStep

        void moveStep(java.lang.Float offset)
        The command accepts a parameter of offset which can be of the range -1.0 to 1.0. Negative values are in one direction and positive values are in the opposite value. For example -1.0 would be maximum offset in a counter clock-wise direction and 0.9 would be 90% offset in a clockwise direction. 0.0 of course would be stop
        Parameters:
        offset - - new offset
